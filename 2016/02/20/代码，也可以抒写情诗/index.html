代码，也可以抒写情诗

2016-02-20 • 标签: blogs

记得曾经问过妈妈：“你说，程序员在你印象里，是什么样的？”“程序员啊，写代码的？挺老实的吧。”

已经记不清是哪一天，曾经听到过有人将“傻、多、早”三个字，作为对程序员评价。

“人傻，钱多，死得早”。这些评价，无不影射出人们对程序员的普遍印象——“情商低”。

也许，他们只是忙于解决bug，无暇去展现自己的浪漫；也许，他们只想把自己的另一面，献给内心中的那个人；也许，他们真的情商低吧。

你说，如果哈姆雷特也会写代码，那么，会有多少个哈姆雷特呢？

你可以找到，三行情诗，写得感天动地的理科生，也可以找到，请假条，写得不知所云的文科生。（这里没有黑谁的意思）

程序员和其他群体一样，有的人眼里，代码只是代码，而有的人眼里，代码，却可以抒写情诗…

1/
while(grass.BearItSeed()&&wind.ShakeItsLeaves() )
{
    we.StandingUp();
    we.Silent();
    everything.Fine();
}
/*
草，
在结它的种子，
风，
在摇它的叶子，
我们，
站着不说话，
就好。
*/
